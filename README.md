This project is developed as part of the Naan Mudhalvan Salesforce Developer Track.
The application, built using Salesforce Lightning, aims to efficiently manage medical inventory by connecting healthcare providers, pharmacies, and donors with hospitals and clinics in need of medical supplies.

Features

Donor registration and medical inventory listing
Hospital/Clinic portal for viewing available medical supplies
Real-time inventory tracking using Lightning components
Automated notification system using Salesforce Flows
Data management through custom objects and relationships
Technologies Used

Salesforce Lightning App Builder
Apex Classes & Triggers
Lightning Web Components (LWC)
Salesforce Data Model & Flows
Objective
To create a sustainable, cloud-based platform that ensures critical medical supplies reach healthcare facilities efficiently, reducing shortages and improving patient care.
